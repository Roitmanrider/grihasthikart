# Purpose

This document defines the official User Interface standards for GrihasthiKart.

Every developer and every AI coding assistant shall follow these guidelines.

The Desktop Homepage and Mobile Homepage stored in:

docs/02_UI_Design/Desktop_Homepage.png

docs/02_UI_Design/Mobile_Homepage.png

are the official design references.

If generated UI differs from these references,

the reference images shall always take priority.

---

# Design Philosophy

The interface should communicate:

Trust

Freshness

Professionalism

Speed

Simplicity

Modern Design

Premium Quality

The interface should never feel cluttered.

Every page should be clean and easy to understand.

# UI Principles

Maintain consistency.

Maintain visual hierarchy.

Minimize clicks.

Keep navigation simple.

Avoid unnecessary animations.

Use white space effectively.

Prioritize readability.

Optimize for mobile.
